# An example for myThings and Belize on Intel Edison

## Setup

```
 $ vi config/default.json
 {
   "meshblu": {
     "uuid": "{uuid of action-1}",
     "token": "{token of action-1}",
     "server": "{ip address of your IDCF server}",
     "port": 80
   },
     "voicetext": {
     "enabled": false,
     "apikey": "{api key}"
   }
 }
```
